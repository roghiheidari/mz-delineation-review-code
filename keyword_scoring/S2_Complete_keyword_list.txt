relevance scoring of 1,038 papers. Keywords were identified through systematic 
analysis of 250 randomly sampled abstracts (random seed: 42).

================================================================================
BONUS SCORING
================================================================================

COMPLETE STUDY BONUS (+5 points)
---------------------------------
Papers receive a bonus if they contain keywords from ALL three categories:
1. Methods (at least one keyword)
2. Data (at least one keyword)
3. Results (at least one keyword)

Rationale: Papers with methods, data, and results represent complete empirical 
studies, which are most valuable for our review.

================================================================================
SCORING ALGORITHM
================================================================================

For each paper:

1. Initialize score = 0

2. For each keyword category:
   - Check if any keywords from category appear in title or abstract
   - If yes, add category weight to score (only once per category)

3. Check for complete study:
   - If has methods AND data AND results keywords
   - Add bonus: +5 points

4. Final score range: -5 to 21 points
   - Minimum: -5 (only exclusion keywords)
   - Maximum: 21 (all categories + bonus)

5. Papers ranked by score (highest to lowest)

6. Top 300 papers selected (score ≥ 18)

================================================================================
KEYWORD DETECTION METHODOLOGY
================================================================================

Text Processing:
----------------
1. Convert title and abstract to lowercase
2. Combine into single text string
3. Search for keyword presence (case-insensitive)

Matching Rules:
---------------
1. Exact substring matching
2. Wildcards (*) match any characters
3. Only one match per category counts (no double-counting)
4. Keywords checked in order listed

Example:
--------
Text: "This study uses k-means clustering to delineate management zones 
       based on yield data and soil electrical conductivity. Results show 
       improved efficiency with R2 = 0.85."

Matches:
- Methods: "k-means", "cluster", "delineat", "management zone" → +5 points
- Data: "yield data", "soil", "electrical conductivity" → +4 points
- Results: "r2", "efficiency", "improved" → +3 points
- Context: "precision agriculture" NOT found → 0 points
- General: "soil" (already counted in Data) → 0 points
- Exclusion: none → 0 points
- Bonus: Has methods + data + results → +5 points

Total Score: 5 + 4 + 3 + 5 = 17 points

================================================================================
LIMITATIONS
================================================================================

1. Keyword-based approach may miss relevant papers with different terminology
2. Scoring weights are somewhat arbitrary (though evidence-based)
3. Does not capture semantic meaning or context
4. May prioritize papers with keyword-rich abstracts
5. Requires manual validation of top-ranked papers

However, this approach:
- Provides systematic, reproducible prioritization
- Reduces reviewer burden for large paper sets
- Is transparent and auditable
- Has been validated through sampling
- Is supplemented by manual screening
