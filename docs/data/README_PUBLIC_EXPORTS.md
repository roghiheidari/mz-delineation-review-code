# Public database export (CSV/JSON)

These files provide a machine-readable export of the **same columns shown in the interactive GitHub Pages table** for the review database associated with:

*The Evolution of Data-Driven Management Zone Delineation: A Systematic Review*

## Files

- `papers_public.csv` — tabular export (one row per paper)
- `papers_public.json` — JSON export (list of records; one object per paper)

## Column definitions (as in the website)

- `Paper_ID`
- `Title`
- `DOI`
- `Year`
- `Country`
- `Crops`
- `Workflow group`
- `Data used for MZ`
- `Methods (workflow explanations)`
- `Validation tier`
- `Validation description`
- `Abstract` (short snippet)
